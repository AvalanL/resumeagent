# Industry Templates Skill

Provide industry-specific resume guidance and templates.

## Trigger

User says `/templates` or asks for industry-specific advice.

## Available Industries

### 1. Tech / Software
**Emphasis:** Skills, projects, impact metrics
**Unique sections:** Tech stack, GitHub, projects
**Format:** 1 page unless 10+ years

```
SUMMARY: Tech-forward, mention key languages/frameworks

SKILLS: (Prominent, near top)
Languages: Python, JavaScript, TypeScript
Frameworks: React, Node.js, Django
Tools: AWS, Docker, Kubernetes, Git

EXPERIENCE: Heavy on technical impact
• Built [feature] using [tech], improving [metric] by [X]%

PROJECTS: (Optional but valuable)
• Open source contributions
• Side projects with links
```

### 2. Finance / Banking
**Emphasis:** Credentials, deal size, precision
**Unique sections:** Certifications (CFA, CPA), licenses
**Format:** Conservative, 1 page

```
SUMMARY: Credentials first, numbers prominent

EXPERIENCE: Deal-focused
• Executed $50M M&A transaction for [industry] client
• Managed portfolio of $XX assets
• Reduced risk exposure by X%

EDUCATION: Prominent (target schools matter)

CERTIFICATIONS: CFA, CPA, Series 7/63, etc.
```

### 3. Healthcare
**Emphasis:** Credentials, patient outcomes, compliance
**Unique sections:** Licenses, certifications, clinical hours
**Format:** Can be 2 pages for clinical roles

```
CREDENTIALS: (Top of resume)
RN, BSN | License #XXXXX | State

CLINICAL EXPERIENCE:
• Patient population and volume
• Outcomes improved
• Protocols implemented

CERTIFICATIONS: BLS, ACLS, specialty certs
```

### 4. Marketing
**Emphasis:** Results, campaigns, creativity + data
**Unique sections:** Portfolio link, notable campaigns
**Format:** 1 page, can be slightly creative

```
SUMMARY: Blend of creative and analytical

EXPERIENCE: Campaign-focused
• Launched campaign generating $XM in revenue
• Grew social following by X% (X to Y)
• Achieved X% conversion rate vs Y% industry avg

SKILLS: Mix of tools and capabilities
Analytics: Google Analytics, Mixpanel
Creative: Adobe Suite, Figma
Platforms: HubSpot, Salesforce Marketing Cloud
```

### 5. Sales
**Emphasis:** Quota attainment, revenue, rankings
**Unique sections:** Notable clients (if allowed)
**Format:** 1 page, numbers-heavy

```
SUMMARY: Revenue-focused, quota achievement

EXPERIENCE: Always include numbers
• Exceeded quota by X% ($Y revenue)
• Ranked #Z out of N sales reps
• Closed largest deal in company history ($X)
• Expanded accounts by X%, adding $Y ARR

SKILLS: CRM tools, methodologies (MEDDIC, Challenger)
```

### 6. Operations / Supply Chain
**Emphasis:** Efficiency, cost reduction, scale
**Unique sections:** Certifications (Six Sigma, PMP)
**Format:** 1 page, process-focused

```
EXPERIENCE: Efficiency-focused
• Reduced costs by $XM through [initiative]
• Improved on-time delivery from X% to Y%
• Managed supply chain for $XM inventory

CERTIFICATIONS: Six Sigma, Lean, PMP, APICS
```

### 7. Creative / Design
**Emphasis:** Portfolio first, tools, style
**Unique sections:** Portfolio link (essential!)
**Format:** Can be creative, but readable

```
PORTFOLIO: [Link] — Make this prominent!

EXPERIENCE: Project-focused
• Redesigned [product] resulting in [metric]
• Led visual identity for [brand/campaign]

TOOLS: Figma, Adobe Creative Suite, etc.

STYLE: Brief note on design philosophy
```

### 8. Education
**Emphasis:** Teaching experience, student outcomes
**Unique sections:** Certifications, subjects, levels
**Format:** Can be 2 pages

```
CERTIFICATIONS: Teaching license, state

TEACHING EXPERIENCE:
• Subjects and grade levels
• Student outcome improvements
• Curriculum developed

PROFESSIONAL DEVELOPMENT: Ongoing learning
```

### 9. Nonprofit / Social Impact
**Emphasis:** Mission alignment, outcomes, fundraising
**Unique sections:** Causes, volunteer work
**Format:** 1-2 pages

```
SUMMARY: Mission-driven language

EXPERIENCE: Impact-focused
• Raised $XM for [cause]
• Served X beneficiaries
• Expanded program to X locations

VOLUNTEER WORK: More prominent than in other industries
```

### 10. Executive / C-Suite
**Emphasis:** Vision, P&L, board experience
**Unique sections:** Board roles, speaking, publications
**Format:** 2 pages acceptable

```
EXECUTIVE SUMMARY: Strategic, vision-focused

EXPERIENCE: P&L and transformation-focused
• Led organization of X employees, $Y revenue
• Drove X% growth / turnaround
• M&A, fundraising, exits

BOARD POSITIONS: If applicable

SPEAKING/PUBLICATIONS: Thought leadership
```

## Choosing the Right Template

Ask:
1. What industry are you targeting?
2. How many years of experience?
3. What level? (Entry, mid, senior, exec)
4. Any unique circumstances? (Career change, gaps)

## Output

Provide:
1. **Template structure** for their industry
2. **Key sections** to include
3. **Language style** (formal, technical, creative)
4. **Example bullets** for their field
5. **What to avoid** in that industry

## Cross-Industry Transitions

If changing industries:
1. Identify transferable skills
2. Learn target industry language
3. Lead with relevant experience
4. Consider functional format
5. Add industry courses/certs
