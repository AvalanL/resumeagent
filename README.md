# ResumeAgent

> Your AI career coach. Modern resumes, compelling cover letters, and interview prep that gets you hired.

## What Is This?

ResumeAgent is an AI-powered career assistant that helps you:
- Create ATS-optimized resumes
- Write personalized cover letters
- Optimize your LinkedIn profile
- Prepare for interviews
- Negotiate better salaries

Open it in **Cursor** (or any AGENTS.md-compatible IDE) and just... talk to it.

## Quick Start

```bash
git clone https://github.com/AvalanL/resumeagent.git
cd resumeagent
# Open in Cursor and start chatting
```

## Commands

| Command | What It Does |
|---------|--------------|
| `/resume` | Create an ATS-optimized resume |
| `/cover` | Write a tailored cover letter |
| `/linkedin` | Optimize your LinkedIn profile |
| `/interview` | Prepare for interviews + practice questions |
| `/salary` | Negotiate your compensation |
| `/pdf` | Export your resume as a PDF |
